
# Materiais

Os materiais utilizados no projeto foram:

Microcontrolador:

- ESP32

Sensores:

- Módulo HX711 (para balança)
- Sensor ultrassônico
- Sensor de temperatura da água

Atuadores:
- Bomba d'água
- Aquecedor 12V
- Servo motor
- Relé
- LEDs RGB

Estrutura:
- Potes de aço inoxidável
- Funil de aço inoxidável
- Suportes de aço inoxidável
- MDF
- Parafusos
